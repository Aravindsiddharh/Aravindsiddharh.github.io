Projects

A Butler Robot for Restaurant Operation

11/2024 – 12/2024

This project is designed for restaurant operations, specifically automating tasks like food delivery from the kitchen to customers' tables. It manages multiple orders efficiently, ensures accurate navigation in a dynamic environment, and handles scenarios like cancellations and timeouts. By integrating ROS for task management, path planning, and obstacle avoidance—alongside control systems for trajectory tracking and feedback-based motion regulation—the robot maintains stability and responsiveness in real time. These enhancements streamline restaurant operations, reduce human effort, and improve service efficiency.

Autonomous Navigation for Tugged Vehicles in ROS2 using nav2

07/2023 – 11/2023

This project aims to develop an advanced autonomous navigation system for an Autonomous Mobile Robot (AMR) tailored specifically for warehouse operations. The system utilizes state-of-the-art Simultaneous Localization and Mapping (SLAM) techniques alongside the nav2 tools within the Robot Operating System (ROS). Integrated control systems such as PID-based velocity and heading controllers ensure smooth and accurate motion, especially in dynamic environments. This cohesive setup enables the AMR to navigate autonomously, dynamically map its surroundings, and maintain precise real-time localization. The project's ultimate goal is to optimize warehouse logistics by improving efficiency, reliability, and safety through robust navigation and motion control capabilities.

Autonomous Navigation of Turtlebot3

03/2021 – 04/2021

This project focuses on implementing autonomous navigation using the TurtleBot3 platform. Utilizing ROS (Robot Operating System) and its navigation stack, the goal is to enable the TurtleBot3 robot to navigate autonomously in indoor environments. Key objectives include developing and tuning navigation algorithms, integrating sensors for obstacle avoidance, and optimizing path planning for efficient traversal. The project aims to demonstrate the feasibility and effectiveness of autonomous robotics in real-world applications, contributing to advancements in robotic navigation.

AI-Based Vision System for Autonomous Navigation in a Self-Driving Toy Car

2024

This study highlights cost-effective, scalable approaches to autonomous systems, advancing intelligent transportation technologies.
