Production-Ready Multi-Robot Fleet Management System

This project delivers a fully integrated, production-ready multi-robot fleet management system built on Open-RMF FreeFleet and ROS 2 (Humble), enhanced with the Eclipse Zenoh-DDS bridge for exceptional scalability and reliability in real-world environments.

Key Features:

  • Multi-robot simulation and deployment in Gazebo
  • Robust localization & Navigation2 with namespaced TF
  • RMF integration (traffic, lifts, doors, maps)
  • Zenoh communication over CycloneDDS
  • Optimized dependency management (NumPy < 2.0)

The system enables scalable coordination of multiple robots with real-time scheduling, efficient communication, and reliable task execution in dynamic environments.
